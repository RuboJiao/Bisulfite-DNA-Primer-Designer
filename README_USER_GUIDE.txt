================================================================================
          BisulfiteDesigner - 甲基化DNA扩增引物可视化设计工具
                           软件功能与使用说明
================================================================================

[ 软件简介 ]
本工具专为重亚硫酸盐（Bisulfite）转化后的DNA序列设计引物而生。通过“六链联动”
可视化技术，帮助研究人员直观地处理复杂的转化序列，精确计算Tm值并评估引物风险。

================================================================================
一、 核心功能亮点
================================================================================

1. 全面的六链展示系统
   - 自动生成转化链：OT (Original Top), OB (Original Bottom)
   - 自动生成互补链：CTOT, CTOB
   - 保留原始参考链：F (Forward), R (Reverse)
   - 实时联动：在一处标注甲基化，六条链同步更新转化结果。

2. 物理与生物信息对齐
   - 序列浏览器采用“所见即所得”的物理对齐，同时在导出与设计时自动处理生物学
     标准的 5' -> 3' 反转，确保引物序列符合实验标准。

3. 精确的热力学计算
   - 基于 NN (Nearest Neighbor) 模型计算 Tm 值。
   - 支持多种盐离子浓度（Na+, Mg++, dNTPs）修正。
   - 特色：支持 LNA (锁核酸) 增强修饰及 MGB (小沟结合物) 修饰的 Tm 增益计算。

4. 深度引物评估
   - 发卡结构 (Hairpin)：自动搜索 3' 末端风险，支持长距离环结构检测。
   - 自身二聚体 (Self-Dimer)：检测引物自身的互补风险。
   - 交叉二聚体 (Cross-Dimer)：支持与项目中已存在的其他引物进行两两比对。

================================================================================
二、 操作指引
================================================================================

1. 导入序列
   - 点击顶部 [导入] 按钮，支持 .fa / .gb / .dna / .txt 格式。
   - 或在首页直接粘贴原始序列。

2. 标注甲基化位点 (mC)
   - 在 DNA 视图中，用鼠标拖拽选中一段包含 'C' 的区域。
   - 点击顶部 [标注 mC] 按钮，该位点将变为红色且在转化链中保持为 'C'。
   - 再次点击可取消标注。

3. 设计引物
   - 选中目标区域后，点击顶部 [添加引物] 按钮。
   - 在弹出的“引物详细设计”窗口中，您可以：
     * 修改引物名称。
     * 编辑引物序列（支持添加 Tail、LNA 或简并碱基）。
     * 实时查看引物与模板的配对可视化（蓝框区域）。
     * 查看 Tm、GC 含量及自由能 (ΔG)。
     * 切换底部标签页检测二聚体与发卡结构。

4. 保存与导出
   - 点击 [保存] 按钮导出 .methdna 格式文件。
   - 该文件保存了所有甲基化标注、引物设计及参数设置，方便下次继续编辑。

================================================================================
三、 符号与书写规范
================================================================================

[ 序列定义 ]
- a, t, c, g : 标准 DNA 碱基（小写）。
- A, T, C, G : LNA (锁核酸) 修饰碱基（大写），将获得额外的 Tm 增益。
- [atgc]     : 括号内的碱基定义为“不与模板配对的额外序列”（如接头、酶切位点）。
- R, Y, S... : 支持 IUPAC 标准简并碱基（自动匹配模板兼容性）。

[ 风险提示 ]
- ΔG (自由能) : 负值绝对值越大表示结构越稳定。
- 警戒线 : 当 ΔG < -2.0 kcal/mol 时，系统会以红色闪烁提醒，表示存在潜在实验风险。

================================================================================
四、 设计建议
================================================================================

1. 针对 Bisulfite 序列的特殊性，建议引物长度在 25-35nt 之间。
2. 尽可能将引物 3' 末端锚定在非 CpG 位点的 'C'（转化后的 'T'）上，以保证扩增
   的特异性。
3. 如果 Tm 值过低，可尝试将中间的 2-3 个碱基替换为大写字母 (LNA 修饰)。
